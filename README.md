## 简介

使用 HybridCLR 对 StarForce 实现热更新。

将 StarForce 的游戏逻辑剔除，可以得到一个接入了 HybridCLR 的 GameFramework [UnityGameFramework](https://github.com/GREAT1217/UnityGameFramework)
